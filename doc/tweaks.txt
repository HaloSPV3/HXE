INITIATION TWEAKS
=================

The HXE kernel permits the tweaking of HCE's initiation. This is done by
applying relevant commands and values to the initc.txt file.

The following tweaks are available for configuration:

  Property       Description                           Notes
  -------------- ------------------------------------- -------------------
  Cinematic      Toggles cinematic bars in cutscenes   SPV3.2 mode only
  Sensor         Toggles motion sensor visiblity       SPV3.2 mode only
  Magnetism      Toggles controller magnetism          
  Auto Aim       Toggles controller auto-aim           
  Acceleration   Toggles mouse acceleration            
  Unload         Toggles shader deactivation           SPV3.2 mode only
